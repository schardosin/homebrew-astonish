# Homebrew Tap for Astonish

This is the official Homebrew tap for [Astonish](https://github.com/schardosin/astonish), an AI-powered terminal assistant.

## Installation

```bash
brew tap schardosin/astonish
brew install astonish
```

## Updating

```bash
brew update
brew upgrade astonish
```

## Uninstallation

```bash
brew uninstall astonish
brew untap schardosin/astonish
```

## Supported Platforms

- **macOS** (Intel and Apple Silicon)
- **Linux** (x86_64 and ARM64)

## More Information

For more information about Astonish, visit the [main repository](https://github.com/schardosin/astonish).
